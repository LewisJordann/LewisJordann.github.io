STRapp - Seizure Technical Responce Application

Java | XML | Python | Arduino

Seizure Technical Response app, STRapp, is a user-friendly Android application that connects to a custom-built wristband via Bluetooth. It allows individuals with epilepsy and other seizure disorders to alert their emergency contacts automatically when a seizure has been detected based on data received from the wristband. The wristband currently includes three sensors, which are electrodermal activity (EDA), accelerometer, and heart rate. Seizure detection is implemented on a backend server that processes data recordings using data mining algorithms. Once a seizure has been detected, a help request is sent from the STRapp. Help requests can also be sent manually if immediate help is needed or if help might be needed soon. Also, if the duration of a seizure is longer than a user-determined time, then emergency services are automatically contacted. Seizures can also be recorded automatically or manually by the user. This allows the user to export this data and forward it to their medical professional. Moreover, the user can view their vitals in real-time. By knowing this information, the individual has the ability to verify their suspicions regarding a likely seizure. STRapp aims to provide peace of mind to individuals who suffer from seizure disorders.

Heart Attack Prediction Analysis

Python PANDAS

Apart of an advanced data mining class, my group was tasked with analysing an existing dataset and analysis on heart attack prediction from kaggle. Following the CRISP-DM, we verified that the original analysis was correct. Then using LIME (Local Interpretable Model-Agnostic Explanations), we analyzed the 7 different models and determined which ones would make the most sense for detecting heart attacks based on the given dataset.
